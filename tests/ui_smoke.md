# UI smoke test

Manual smoke checklist for the KanBee frontend.

1. Open the site in a browser.
2. Register a new user or sign in with an existing user.
3. Create a task with a non-empty title, status and priority.
4. Drag the task to another Kanban column.
5. Switch the theme in the settings menu.
6. Refresh the page and confirm the task and selected theme are still visible.
